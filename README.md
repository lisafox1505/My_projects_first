# 🚀 Навчальні проєкти на Python: Чат-Бот та Гра "Мільйонер"

Цей репозиторій містить два комплексні проєкти по курсу Python Starter, що демонструють володіння мовою Python.

---

## 🤖 Проєкт №1: Розважальний консольний Чат-Бот

Багатофункціональний консольний помічник, написаний на Python. Бот вміє рекомендувати медіаконтент, розповідати жарти, ділитися рецептами, загадувати загадки та пропонувати зіграти в повноцінні міні-ігри. Проєкт демонструє принципи ООП, декомпозиції, роботу з файловою системою (JSON) та багатопотоковістю.

## ✨ Основний функціонал

* 🎬 **Підбірки контенту:** Пошук фільмів, музики та ігор за жанрами, виконавцями або роком випуску.
* 😂 **Розваги:** Генерація випадкових анекдотів та цікавих фактів.
* 🍳 **Кулінарія:** Виведення покрокових смачних рецептів з інгредієнтами.
* 🧠 **Загадки:** Інтерактивна система загадок з обмеженням у 5 спроб.
* 🎨 **Стильний інтерфейс:** Використання ANSI-кольорів, прогрес-барів (tqdm) та ASCII-артів для приємного UX у терміналі.

## 🎮 Вбудовані міні-ігри

Проєкт включає 3 повноцінні ігри, які винесені в окремі модулі:

1. **Битва з монстром (RPG):** Текстовий бій, де потрібно стежити за HP, використовувати атаки та зілля лікування.
2. **Таблиця множення (Math Trainer):** Гра на час із використанням `threading`. У тебе є рівно 1 хвилина на кожен з 10 раундів, та 3 життя, щоб вирішити максимум прикладів!
3. **Пайтон - Казино (Slot Machine):** Симулятор ігрового автомата з розрахунком ймовірностей, системою ставок та джекпотом.

## 🛠 Технології та бібліотеки

* **Мова:** Python 3.10+
* **Стандартні бібліотеки:** `json`, `random`, `time`, `os`, `sys`, `threading`
* **Сторонні модулі:**
    * `colorama` — для кольорового виведення в консоль.
    * `tqdm` — для анімації завантаження.
    * `art` — для генерації ASCII-тексту.
    * `emoji` — для підтримки смайликів у терміналі.

## 💎 Проєкт №2: Гра "Хто хоче стати мільйонером?" (Pygame)

Графічна адаптація інтелектуального шоу "Хто хоче стати мільйонером?".

### 🎮 Основні можливості:
* **Графічний інтерфейс:** Створено за допомогою `pygame` (вікна привітання, поле введення імені, основне ігрове поле та екрани перемоги/програшу).
* **Інтелектуальна логіка підказок:**
    * `50/50`: Автоматично видаляє дві випадкові неправильні відповіді.
    * `Дзвінок другу`: Пропонує варіант відповіді на основі поточного списку кнопок.
    * `Допомога залу`: Генерує статистичну ймовірність для кожного варіанту, враховуючи правильну відповідь.
* **Ігрова механіка:** 15 рівнів питань із прогресивною шкалою виграшу, "незгораємими" сумами та системою затримок для реалістичності.
* **Музичний супровід:** Фонова музика та звукові ефекти для створення атмосфери шоу.

### 🛠 Технології та бібліотеки:
* **Мова:** Python 3.10+
* **Графічний движок:** `pygame` — використаний для створення вікна гри, малювання кнопок та обробки натискань миші.
* **Робота зі звуком:** `pygame.mixer` — для програвання фонової музики.
* **Робота з даними:** `json` — для зберігання та завантаження бази питань.
* **Логіка:** `random` — для випадкового вибору питань та роботи підказок.

## 📂 Структура проєкту

Для зручності проєкт розділений на дві основні частини:

1. **Папка з Чат-ботом:**
    * `data_file/` — файли JSON з фільмами, рецептами та жартами.
    * `all_games/` — окремі файли з кодом для міні-ігор (монстр, математика, казино).
    * `main.py` та `classes.py` — головні файли, що запускають бота.

2. **Папка з грою "Мільйонер":**
    * `main_game.py` — основний файл, який запускає гру.
    * `functions.py` — файл, де прописана логіка підказок та перемикання питань.
    * `constans.py` — список кольорів, розмірів кнопок та налаштувань екрана.
    * `questions.json` — список усіх питань для вікторини.
    * `music.mp3` та `logo.png` — музика та картинки для оформлення.

* `requirements.txt` — файл зі списком бібліотек, які потрібно встановити для запуску обох проєктів.
